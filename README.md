# Blue

A lightweight community tool to help people find clean water stations during Mardi Gras.

## 🚀 Getting Started

You can open `index.html` directly in your browser or use a simple static server:

```bash
# Python 3 (recommended)
python -m http.server 8000

# Node.js (if you have it)
npx serve -s .
```

Then open: http://localhost:8000

## 📍 What’s Included

- ✔️ **Interactive map** showing shared clean water stations
- ✔️ **Quick reporting form** to add a new station (with rating + optional photo)
- ✔️ **Ratings** so the community can surface the most accessible, clean sources
- ✔️ **Chat-style comments** so people can ask about availability/limits
- ✔️ **Geolocation support** to center on your location
- ✔️ **Responsive mobile-first design** for on-the-go usage

## 🛠️ Customize

- Update markers and station data in `script.js`
- Adjust styles in `styles.css`
- Add new sections or improve the map experience in `index.html`
